# Disambiguation

# Game Boy Advance

Game Boy Advance development is covered by another project, the awesome-GBAdev (opens new window) list. GBA, however, can run GB/GBC games. It does so in a slightly different way compared to native hardware. This is covered in the Emulator Development section of this list.

# Game Boy Color and Super Game Boy

This list is focused on the original (1989) Game Boy (DMG), the Game Boy Color (GBC) and Super Game Boy (SGB) are very similar systems, with a few important distinctions, such as:

  • Different hardware specifications
  • Specific hardware and software features
  • Specific registers
  • Specific bugs, quirks and exploitable behaviours

If you aim to develop your software for SGB or GBC, or you want to know how it runs on the other systems, you may want to take advantage and adapt to these differences, check the Game Boy Color category and look for specific references to GBC/CGB and SGB.

# Software Development

The To C Or Not To C (opens new window) essay gives an overview of the Game Boy's capabilities, discussing the pros and cons of the available development tools, and providing tips to write more efficient code.

# Assemblers

  • RGBDS (opens new window) - Assembler and linker package. Documentation (opens new window).
  • ASMotor (opens new window) - Assembler engine and development system targeting Game Boy, among other CPUs. Written by the original RGBDS author. Documentation (opens new window).
  • wla-dx (opens new window) - Yet Another GB-Z80/Z80/... Multi Platform Cross Assembler Package. Documentation (opens new window).

# Compilers

  • GBDK (opens new window) - A set of tools that enable to develop in C. Includes a set of libraries for the most common requirements and generates ready ROM files. Last updated in 2001. Documentation (opens new window).
    • GBDK-2020 (opens new window) - A modernized GBDK with updated SDCC toolchain (supports ROM banking), bug fixes, documentation and other improvements. Supports Windows, Linux and MacOS.
    • gbdk-osx (opens new window) - Patched GBDK 2.96a for the latest compilers on macOS.
    • gbdk-n (opens new window) - Aims to update the gbdk libraries to be compatible with new versions of SDCC and provide helpers for building roms. Does not support ROM banking, thus limiting ROM size to 32K.

# Experimental/Proof of Concepts

  • Wiz (opens new window) - A high-level assembly language for writing homebrew on retro console platforms (Game Boy, NES, Atari 2600, and more).
  • gbforth (opens new window) - A Forth-based Game Boy development kit.
  • gbasm-rs (opens new window) - An opinionated Rust based compiler for Game Boy z80 assembly code.
  • gbasm (opens new window) - A JavaScript based compiler for Game Boy z80 assembly code.
  • tniASM (opens new window) - Macro Assembler.
  • Assembler (opens new window) - Assembler written in Swift.
  • llvm-gbz80 (opens new window) / clang-gbz80 (opens new window) - Clang/LLVM port to the GBZ80 CPU (similar to the deprecated euclio/llvm-gbz80 (opens new window)).
  • gbdk-go (opens new window) - A compiler translates Go programs to C code. The output C code is built into GB ROM by GBDK.

# Emulators

  • BGB (opens new window) - Powerful emulator and debugger. Provides an accurate hardware emulation.
  • SameBoy (opens new window) - Accurate emulator with a wide range of powerful debugging features.
  • Mooneye GB (opens new window) - Research project and emulator in Rust.
  • mGBA (opens new window) - Modern cross platform GBA emulator which also runs GB/GBC games.
  • Binjgb (opens new window) - 5Kloc emulator that passes most of the tests. *Rewind- feature. Runs in the browser using WebAssembly.
  • Gambatte (opens new window) - Cross-platform and accurate emulator.
  • MetroBoy (opens new window) - A playable, circuit-level simulation of an entire Game Boy.
  • gbe-plus (opens new window) - A recently rewritten emulator that has a large effort in preserving the functions of obscure accessories (such as IR link, Mobile Network GB, Barcode Boy, GB Printer, local and online GB Serial Link Cable, ... )
  • Emulicious (opens new window) - Provides accurate emulation and includes powerful tools such as a profiler and source-level debugging for ASM and C via a VS Code debug adapter (opens new window).
